Requirements

Java version 7. Java can be found at www.java.com/getjava.

How to use

Setting up

1. Open Steam and click on Library.
2. Right click on Dota 2, and select Properties.
3. Select "Set Launch Options".
4. Add "-override_vpk" (without quotes) to the launch options.
5. Click Ok.

Running the program

Simply open Dota_Sound_Editor.jar!

Using Dota 2 Sound Editor

The Dota 2 Sound editor allows you to modify the sounds that are played when heroes in Dota 2 perform actions. To begin, select a hero from the dropdown box. Then, expand the list of sounds associated with a particular action. From there, you can view each sound that may occur when a hero performs that action. For each sound, you can:

Replace
This allows you to change what sound plays when the selected action is performed.

Revert
This restores the selected action to its default sound.

Revert All
This restores all actions for to current hero to their default sounds.

Advanced/Basic
The Advanced button allows you to view the current hero's entire sound script as a tree. Triple-click any node to allow manual editing.
While in Advanced mode, Replace, Revert and Play Sound will only work on nodes that contain a path to a sound file.
Clicking the Basic button will return the editor to Basic mode.
